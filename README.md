# Awesome-ComputerScience-JobSearch-Toolkit

本仓库是计算机相关专业的求职工具，提供计算机相关专业（计算机科学与技术、计算机技术、人工智能、软件工程、网络空间安全、网络与信息安全等）的求职信息，包括但不限于：实习生和博士后的招聘公告，硕士和博士的招生等。
